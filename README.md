# Статистический анализ данных о чаевых (CI & P-value)

Проект посвящен проверке гипотез, влияющих на размер оставляемых чаевых, на основе датасета Tips.

Основные моменты:
* Исследование зависимости размера чаевых от пола посетителя и факта курения.
* Расчет доверительных интервалов, проведение t-теста и интерпретация полученных значений p-value для оценки статистической значимости различий.
* Построение столбчатых диаграмм и ящиков с усами для наглядного сравнения распределений в группах.
